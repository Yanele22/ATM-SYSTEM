🏦 Yaya Python ATM System

📖 Overview

Yaya Python ATM System is a console-based banking application developed in Python. It simulates the core functionality of an Automated Teller Machine (ATM), allowing users to securely manage their accounts through an interactive command-line interface.

The project was created to strengthen my understanding of Python programming, object-oriented programming (OOP), file handling, input validation, and version control using Git and GitHub.

This project was developed by **Yanele Bhengu** with the assistance of AI as a learning tool. AI was used to explain programming concepts, provide guidance, suggest improvements, and assist with debugging, while I designed, implemented, tested, and refined the application throughout its development.

✨ Features

* 🔐 Secure login using Account Number and PIN
* 👤 Multiple user accounts
* 💰 Check account balance
* ➕ Deposit money
* ➖ Withdraw money
* 💸 Transfer money between accounts
* 📜 View transaction history with timestamps
* 🔑 Change account PIN
* 👤 View account details
* 🔄 Logout and log in as another user
* ✅ Input validation and error handling
